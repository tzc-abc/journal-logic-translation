# Input Template

Use this file when the user has not supplied enough information for target-journal-aligned translation. Ask only for missing items that affect the requested passage.

## Minimum Intake

```text
Target journal:
Article type:
Section to translate:
Chinese source text:
Study type/design:
Main finding:
Primary data or result supporting the main finding:
Statistics/effect sizes available:
Figures or tables linked to this passage:
Claims that must not be changed:
Journal instructions or model article supplied:
```

## Quick Draft Intake

Use this lighter intake for early drafts and paragraph-level requests:

```text
Target journal or broad style:
Section type:
Chinese paragraph:
Known data that must be preserved:
```

If any field is missing but the passage is low-risk, proceed with a provisional translation and mark assumptions under `Needs author confirmation`.

## Fast Clarifying Questions

Ask these in order when needed:

1. What is the target journal and article type?
2. Is this an abstract, introduction, results, discussion, or full manuscript section?
3. What data directly support the strongest claim in this passage?
4. Are there exact statistics, sample sizes, groups, timepoints, and figure/table numbers that must be preserved?
5. Are any claims speculative, preliminary, or not yet validated?

## Do Not Wait For Everything

Proceed with a safe partial translation when:

- The target journal is known.
- The passage is descriptive or methods-focused.
- Missing details do not affect the translated claim.
- Unsupported claims can be bracketed as author queries.

Stop before translating when missing details would force the model to invent article structure, results, statistics, methods, citations, or journal-required declarations.
